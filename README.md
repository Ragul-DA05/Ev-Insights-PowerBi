⚡ Electric Vehicle Market Insights Dashboard


📌 Overview

This project showcases an interactive Power BI dashboard built to analyze the Electric Vehicle (EV) market.

It provides insights into:

- Sales performance 📈
- Pricing trends 💰
- Vehicle range & charging ⚡
- Manufacturer & country analysis 🌍

The goal is to help stakeholders make data-driven decisions in the rapidly growing EV industry.

---

🎯 Objectives

- Analyze EV sales trends over time
- Compare manufacturers based on performance
- Identify top EV-producing countries
- Evaluate key factors like range, price, and charging time

---

📊 Dashboard Features

🔹 Key KPIs

- Average Price: 90.61K
- Total Units Sold: 31M
- Average Range: 349.90 km
- Average Charging Time: 6.21 hrs

🔹 Visual Insights

- 📈 Year-wise sales trend
- 🏭 Top manufacturers by EV sales
- 🌍 Country-wise production analysis
- 🎨 Popular EV colors distribution
- ⚖️ Multi-metric comparison (price vs range vs charging)
- 🛡️ Safety demand trends

---

🛠️ Tech Stack

- Power BI – Dashboard development
- Excel / CSV – Data source
- Power Query – Data cleaning & transformation
- DAX – Calculated measures & KPIs
